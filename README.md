# Diagnosis DOC API

## Description

Simple API to provide Cases and Conditions to get Diagnosis from Docs!

## Dependecies

- Node
- Docker
- Docker-compose
- Nestjs

## Installation

```bash
$ npm install
```

## Running the app

Run the MongoDB container:

```bash
docker-compose up mongo-diagnosisdoc
```

Run the server:

```bash
# development
$ npm start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
